# Audit

客户端追踪 SDK ，通过获取设备指纹等来标记用户防止业务被薅羊毛，恶意爬虫等影响。

## 设计目的

1.为更多的前端开发者提供标记用户的能力。

2.增加风控防止被薅羊毛和恶意爬虫影响。

3.作为一款开源产品帮助更多的项目提升安全性。

## 项目架构

首先v0.1.0版本设计目的是可以具备客户端追踪的能力以及模拟器识别，并做到异常上报。

```
├── config // webpack 配置文件
├── dist   // 打包输出文件目录
├── server // server
├── src    // 项目文件目录
├── test
```
